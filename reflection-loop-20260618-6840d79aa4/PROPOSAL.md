# Reflection Loop — reflection-loop

## Триггеры

«не успел / сорвал / провалил», `missed_count >= 2`, streak потерян 3+ дней.

## Меню причин (НЕ открытый вопрос)

1. Не хватило времени — план нереалистичный
2. Не хватило сил/фокуса
3. Не понял тему — нужен другой источник
4. Были срочные другие задачи
5. Отвлёкся
6. Забыл про дедлайн
7. Болел / форс-мажор
8. Свой вариант

## Адаптация (причина → действие)

| Причина | Действие |
|---|---|
| План нереалистичен | study-plan: меньше часов/нед |
| Тема непонятна | Разбить на подтемы, материалы |
| Приоритет не тот | Переставить |
| Отвлечение | Включить morning/evening brief |
| Забыл | Добавить milestone reminder |
