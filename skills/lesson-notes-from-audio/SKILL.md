# lesson-notes-from-audio

## Цель
Принимает голосовое сообщение (или текст диктовки) с урока → возвращает структурированный конспект: тезисы, термины с определениями, action items. Использует OpenAI Whisper API для ASR (если голосовое) + Claude/GPT для саммари.

## Триггер
- **Авто:** когда пользователь в Telegram-боте `@Goldenteam239bot` отправляет голосовое с пометкой «урок» (или в `users/<user_key>/profile.md:audio_kinds = ["урок"]`).
- **Вручную:** команда «сделай конспект голосового» / «обработай диктовку» в golden-hour сессии.

## Логика
1. Получить аудио (Telegram voice file_id) или текст (если уже расшифровано).
2. **ASR:** если голосовое — `curl -X POST https://api.openai.com/v1/audio/transcriptions -H "Authorization: Bearer $OPENAI_API_KEY" -F file=@<ogg> -F model=whisper-1 -F language=ru`. Результат: полный текст.
3. **Структурирование:** отправить текст в LLM (через MiniMax API) с промптом:
   ```
   Ты — ассистент для конспектирования уроков. Из транскрипта урока сделай:
   1. **Тезисы** (5-10 пунктов, по существу)
   2. **Термины** (со определениями)
   3. **Action items** (что сделать / повторить / выучить)
   4. **Связанные темы** (если есть)
   Формат: markdown. Язык: русский.
   ```
4. **Сохранить** результат в `users/<user_key>/notes/lesson-YYYY-MM-DD-HHMM.md` (новый файл).
5. **Добавить** action items в `users/<user_key>/tasks.md` (через `current-tasks`).
6. **Сообщить** пользователю: «Конспект сохранён в `notes/lesson-<YYYY-MM-DD-HHMM>.md`, N тезисов, M action items (добавлены в задачи)».

## Вход / Выход
- **Вход:** голосовое (.ogg) ИЛИ текст.
- **Выход:** markdown-конспект + action items в задачах.

## Примеры
### Пример 1
**Запрос:** (голосовое 90 сек про теорему Пифагора)
**Результат:** `notes/lesson-2026-06-17-1440.md`:
```markdown
# Конспект урока 2026-06-17 14:40

## Тезисы
- В прямоугольном треугольнике квадрат гипотенузы равен сумме квадратов катетов.
- Геометрическое доказательство через площади квадратов.
- Применение: расстояния, навигация, строительство.

## Термины
- **Гипотенуза** — сторона треугольника, лежащая напротив прямого угла.
- **Катет** — сторона, прилежащая к прямому углу.

## Action items
- Решить задачи 1-5 из учебника
- Повторить доказательство теоремы
- Подготовить 3 примера из жизни
```

## Что НЕ делает
- ❌ Не распознаёт голос без OPENAI_API_KEY (требует ключ в env).
- ❌ Не обрабатывает длинные лекции (> 30 мин) — лимит Whisper API.
- ❌ Не делает перевод на другие языки (только русский).
- ❌ Не хранит сами аудиофайлы (только текст транскрипта).

## Зависимости
- skills: `note-to-file`, `current-tasks`, `user-profile`
- tools: `exec` (curl), LLM-вызов
- внешние: OpenAI API (OPENAI_API_KEY), MiniMax API
- данные: `users/<user_key>/notes/`, `users/<user_key>/tasks.md`
