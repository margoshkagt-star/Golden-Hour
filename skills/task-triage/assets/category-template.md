# Категории задач — шаблон

> Этот файл заполняется автоматически при первом запуске скилла `task-triage` и пополняется по мере появления новых категорий.

## Формат записи

```markdown
## <emoji> <имя_категории>
- keywords: [триггер1, триггер2, ...]
- boost: <-2..+2>
- created: <YYYY-MM-DD>
- notes: <опционально>
```

## Пример

```markdown
## 🏢 работа
- keywords: [работа, офис, начальник, заказчик, клиент, митинг]
- boost: 0
- created: 2026-06-17

## 💼 проект:openclaw-assistant
- keywords: [фельпик, ассистент, скилл, агенты]
- boost: +2
- created: 2026-06-17
- notes: Стратегический проект хозяина.

## 🏥 здоровье
- keywords: [спорт, врач, анализы, сон, тренировка, зал]
- boost: +1
- created: 2026-06-17
- notes: Пользователь склонен откладывать.
```
