# mood-checkin

## Цель
Расширить `daily-study-checkin` (или `goal-checkin-notifier`) блоком оценки настроения (1-5) после каждой сессии. Писать в `users/<user_key>/progress.md` строку вида `- YYYY-MM-DD HH:MM — mood: 4 — note: "сложно, но интересно"`. Считать streak (дней подряд с mood >= 3).

## Триггер
- **Авто:** после `daily-study-checkin` evening check-in (после закрытия всех задач дня).
- **Вручную:** команда «/mood» / «оцени настроение» в golden-hour сессии.

## Логика
1. После завершения вечернего чек-ина (когда пользователь нажал «всё готово»), `goal-checkin-notifier` отправляет сообщение:
   ```
   🌙 Чек-ин за сегодня завершён.
   Как настроение? (1-5)
   1 — 😣 плохо
   2 — 😐 так себе
   3 — 🙂 нормально
   4 — 😊 хорошо
   5 — 🤩 отлично
   Можно добавить короткую записку (до 200 символов).
   ```
2. Если пользователь отвечает числом 1-5 (с опциональной запиской), записать в `users/<user_key>/progress.md`:
   ```
   ## Mood log
   - 2026-06-18 22:14 — mood: 4 — "сложно, но интересно"
   - 2026-06-17 21:50 — mood: 3 — ""
   ```
3. **Streak:** в `users/<user_key>/progress.md` хранить счётчик:
   - Если сегодня mood >= 3 и вчера mood >= 3 → streak++
   - Если сегодня mood < 3 или пропуск → streak = 0
   - Если сегодня нет записи (пропуск чек-ина) → streak не обновляется (не сбрасывается, ждём)
4. **Утренний показ:** `goal-checkin-notifier` в `morning_brief` добавляет строку:
   ```
   🌅 Настроение: streak=N дней, последний: 4
   ```
5. **Корреляция (опционально):** раз в неделю показывать в evening check-in мини-аналитику: средний mood за неделю, корреляция mood с количеством закрытых задач.

## Вход / Выход
- **Вход:** число 1-5 + опциональный текст (от пользователя).
- **Выход:** строка в `progress.md` + обновлённый streak.

## Примеры
### Пример 1
**Запрос:** (после чек-ина) пользователь отвечает «4»
**Результат:** в `progress.md` добавлено:
```
- 2026-06-18 22:14 — mood: 4
```

### Пример 2
**Запрос:** (после чек-ина) пользователь отвечает «2 устал, не выспался»
**Результат:** в `progress.md` добавлено:
```
- 2026-06-18 22:15 — mood: 2 — "устал, не выспался"
```
Streak сбрасывается в 0 (если до этого был >= 1).

## Что НЕ делает
- ❌ Не делает уведомлений, если пользователь не заканчивал чек-ин.
- ❌ Не показывает streak, если в системе меньше 3 дней записей.
- ❌ Не агрегирует данные с другими пользователями (приватно).
- ❌ Не делает предположений о причине низкого mood (не лезет в душу).
