# help-menu — что умеет бот

Единая точка, где пользователь узнаёт обо **всех** функциях. Бот не прячет возможности: показывает меню после настройки, по запросу и подсказывает релевантное в ходе работы.

## Триггеры
- Слова: «что умеешь», «что ты умеешь», «помощь», «меню», «функции», «возможности», «команды», «/help», «?».
- Авто: один раз сразу после завершения настройки (`setup-finalize` шаг 7).
- Контекстно: если пользователь явно ищет функцию («а ты можешь …?») — ответить из этого меню.

## Поведение по фазе
- **`setup_status ≠ complete`** — показать короткую версию: «Сейчас идёт настройка. После неё открою: план, задачи, фокус-таймер, напоминания, статистику. Допройдём настройку?»
- **`setup_status: complete`** — показать полное меню (ниже).

## Полное меню (после настройки)

```
🌅 Вот что я умею, <name>. Просто напиши словами — пойму.

📅 ПЛАН
• «составь план» — макро-план до дедлайна по приоритетам
• «спланируй день» — сбалансированный день (важное вперёд, без перегруза)
• «неделя 3» — задание на конкретную неделю плана
• «делай каждый день / по будням …» — повторяющиеся дела
• «что повторить» — повтор слабых тем по интервалам (1→3→7→14→30)

✅ ЗАДАЧИ
• «список задач» / «добавь задачу» / «закрой задачу»
• «разбей задачу …» — крупную дроблю на шаги (первый <15 мин)
• «что горит» / «прогресс» — статус и дедлайны
• «… важнее / менее важно», «… тяжелее / легче» — настройка приоритета и сложности

👥 КОМАНДА (общие цели)
• «создай команду …» / «мои команды» — командная работа над одной целью
• «пригласи @… в команду» — инвайт (код на 5 дней)
• «вступить по коду …» — принять инвайт
• «добавь командную задачу …» / «беру» / «сдал» / «принято» — lifecycle таски
• «кто что делает» — доска команды (только если ты в ней)

⏱ ФОКУС И ДИСЦИПЛИНА
• «начать 45 / час» — фокус-таймер, в конце похвалю и зачту
• «чек-ин» / «итог дня» — отметить, что сделал, веду streak
• утренний бриф, пинги по задачам, вечерний чек-ин — напоминания (》«жёстче»/«мягче»)
• «не успел / провалил» — разберём причину и поправим план, без самобичевания

📚 МАТЕРИАЛЫ
• «дай задачу/теорию по …» — выдам из базы или найду
• «сохрани это к цели» — сложу в твою библиотеку
• пришли аудио урока — сделаю конспект (если доступна расшифровка)

📊 СТАТИСТИКА
• «статистика за неделю/месяц/год/всё время» — прогресс в часах и по весу
• «сколько занимался» — время по таймеру (сегодня/неделя/…)

🔗 ИНТЕГРАЦИИ
• «подключи календарь» — двусторонняя синхронизация с Google Calendar
• погода — строкой в утреннем брифе (по твоему городу)

⚙️ НАСТРОЙКИ
• «сделай день легче/тяжелее» — нагрузка
• «короче/длиннее» — длина ответов
• «настроить заново» — пересобрать профиль
• «что умеешь» — это меню

Скажи, с чего начнём?
```

## Правила
- Показывать **только доступное в текущей фазе** (не дразнить рабочими функциями во время настройки).
- Не вываливать длинный список без нужды: если пользователь спросил про конкретную область («а с задачами что?») — показать только её блок.
- Каждый блок = реальные команды, которые ведут на конкретный скилл (см. таблицы в `SOUL.md` → «Рабочий режим»).
- Тон — короткий, дружелюбный, мобильный. Эмодзи 🌅 в начале.
