# exam-topics

## Цель
Зафиксировать **какие темы входят в экзамен** — по кодификатору (ЕГЭ/ОГЭ) или по словам пользователя (вступительные / другой). Без списка тем — не переходить к самооценке.

## Триггер
- После `exam-subject`
- До `exam-self-assess`

## Логика
1. Прочитать `users/<user_key>/profile.md` → `exam_type`, `exam_subject`, `exam_subject_variant`
2. Показать **список тем экзамена** (карта ниже) или попросить пользователя перечислить
3. Пользователь подтверждает список целиком, убирает лишнее или дописывает свои темы
4. **Записать в `users/<user_key>/profile.md` → `exam_topics: [тема1, тема2, ...]`**
5. Если пользователь дал свой список — `exam_topics_source: user|codifier|mixed`
6. Подтвердить и перейти к `exam-self-assess`

## Формат сообщения (общий)

> 📋 Вот **темы**, которые обычно входят в [exam_type] по [предмет]. Отметь, что актуально для тебя:
>
> [нумерованный список тем]
>
> Можешь:
> - написать **«все»** — если готовишься по полному кодификатору
> - перечислить **номера** тем, которые нужны (например: `1, 3, 5-7`)
> - **дописать** свои темы, если чего-то нет в списке
> - для своего экзамена — просто **перечислить темы** списком

## Карты тем (exam_type + exam_subject)

### ЕГЭ — математика профиль (`ege` + `math` + `profile`)

1. Алгебра: уравнения, неравенства, системы
2. Функции и графики
3. Производная и исследование функций
4. Первообразная, интегралы, площади
5. Геометрия (планиметрия, стереометрия)
6. Вероятность и статистика
7. Задачи с параметром
8. Задачи повышенной сложности (12–18)

### ЕГЭ — математика база (`ege` + `math` + `base`)

1. Числа и вычисления
2. Алгебраические выражения и уравнения
3. Функции и графики
4. Геометрия
5. Вероятность и статистика
6. Прикладные задачи

### ОГЭ — математика (`oge` + `math`)

1. Числа и арифметика
2. Алгебраические выражения
3. Уравнения и неравенства
4. Функции и координаты
5. Геометрия
6. Вероятность и статистика

### ЕГЭ — физика (`ege` + `physics`)

1. Механика
2. Молекулярная физика и термодинамика
3. Электродинамика
4. Колебания и волны
5. Оптика
6. Квантовая физика (базово)

### ЕГЭ — русский (`ege` + `russian`)

1. Орфография
2. Пунктуация
3. Лексика и стилистика
4. Средства выразительности
5. Анализ текста
6. Сочинение (аргументация)

### ЕГЭ — информатика (`ege` + `informatics`)

1. Системы счисления, логика
2. Алгоритмы и программирование
3. Массивы, строки, файлы
4. Рекурсия, сортировки
5. Графы и деревья
6. Динамическое программирование

### Вступительные / другой (`admission` | `other`)

> 📋 **Какие темы** будут на твоём экзамене? Перечисли списком — хоть коротко, хоть подробно.

## Поведение

| Ответ | Куда записать |
|---|---|
| `все` / `полный кодификатор` | весь список карты → `exam_topics` |
| Номера (`1, 3, 5`) | выбранные темы → `exam_topics` |
| Свой список тем | дословно в `exam_topics`, `exam_topics_source: user` |
| Карта + дополнения | объединить → `exam_topics_source: mixed` |

## Данные
- `users/<user_key>/profile.md` → `exam_topics: [...]`, `exam_topics_source: codifier|user|mixed`

## Зависимости

- Перед `exam-self-assess`
