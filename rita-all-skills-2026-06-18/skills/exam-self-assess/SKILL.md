# exam-self-assess

## Цель
Проверить **уровень пользователя по каждой теме** из `exam_topics`. Адаптировать форму под длину списка.

## Триггер
- После `exam-topics`
- До `setup-finalize`

## Логика
1. Прочитать `users/<user_key>/profile.md` → `exam_topics`
2. Если тем **≤ 4** — спросить уровень по каждой отдельно
3. Если тем **5–8** — показать таблицу + шкалу, пройтись по всем
4. Если тем **> 8** — сначала «3–5 самых слабых?», потом детально + общий уровень по остальным
5. **Записать в `users/<user_key>/profile.md` → `exam_topic_levels: {тема: уровень, ...}`**
6. Опционально `exam_level_note`
7. Подтвердить и перейти к `setup-finalize`

## Шкала уровня

| Код | Вариант |
|---|---|
| `zero` | **С нуля** |
| `weak` | **Слабо** |
| `medium` | **Средне** |
| `strong` | **Уверенно** |
| `expert` | **Отлично** |

## Формат сообщения (≤ 4 тем)

> 🎯 По **каждой теме** — какой у тебя уровень?
> Шкала: **с нуля / слабо / средне / уверенно / отлично**
> 1. **[тема 1]** — ?
> 2. **[тема 2]** — ?
> Напиши в одном сообщении, например: `1-слабо, 2-средне, 3-уверенно`

## Формат сообщения (5–8 тем)

> 🎯 Оцени себя по темам экзамена (шкала: **с нуля → отлично**):
> [список тем]
> Пройдись по **всем** номерам в одном сообщении.

## Формат сообщения (> 8 тем)

> 🎯 Тем много. Сначала — **3–5 самых слабых**.

## Поведение

| Ответ | Куда записать |
|---|---|
| `1-слабо, 2-средне` | `exam_topic_levels: {"тема1": "weak", ...}` |
| «везде слабо» | все темы → `weak` |
| Свободный текст | `exam_level_note` дословно |

## Данные
- `users/<user_key>/profile.md` → `exam_topic_levels: {тема: zero|weak|medium|strong|expert|unknown}`
- опционально `exam_level_note`

## Зависимости
- Зависит от `exam-topics`
- Перед `setup-finalize`
