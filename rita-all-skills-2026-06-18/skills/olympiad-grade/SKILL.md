# olympiad-grade

## Цель
Уточнить класс пользователя для олимпиадной подготовки. Без класса — не продолжать.

## Триггер
- **Сразу после `purpose-select`**, если `purpose = "olympiad"`
- **До** `olympiad-subject` и `olympiad-self-asses`

## Логика
1. Показать приглашение с вариантами
2. Получить ответ (число 7–11 или «выпускник»)
3. **Записать в `users/<user_key>/profile.md` → `grade: <число или "graduate">`**
4. Подтвердить запись одной строкой
5. Перейти к `olympiad-subject`

## Формат сообщения

> 🎓 Олимпиадная подготовка. Какой у тебя **класс**?
>
> 1. **7 класс**
> 2. **8 класс**
> 3. **9 класс**
> 4. **10 класс**
> 5. **11 класс**
> 6. **Уже не учусь** (выпускник, самоучка)
>
> Напиши цифру или слово — что угодно.

## Поведение

| Ответ | grade | Реакция |
|---|---|---|
| `1` / `7` / `седьмой` | `7` | `Записал: 7 класс.` |
| `2` / `8` / `восьмой` | `8` | `Записал: 8 класс.` |
| `3` / `9` / `девятый` | `9` | `Записал: 9 класс.` |
| `4` / `10` / `десятый` | `10` | `Записал: 10 класс.` |
| `5` / `11` / `одиннадцатый` | `11` | `Записал: 11 класс.` |
| `6` / `выпускник` / `взрослый` / `самоучка` | `graduate` | `Записал: выпускник.` |
| Другое | — | Переспросить мягко, не осуждать |

## Что НЕ делает
- Не переходит к предмету, пока класс не записан
- Не оценивает выбор («рано для олимпиад» — никогда)
- Не выбирает класс за пользователя (переиспользование старого профиля решает `session-start`)

## Данные
- `users/<user_key>/profile.md` → `grade: <7|8|9|10|11|graduate>`

## Зависимости
- Зависит от `purpose-select` (`purpose` = `olympiad`)
- Перед `olympiad-subject`
