# exam-topics

## Цель
Зафиксировать **какие темы входят в экзамен** — по кодификатору (ЕГЭ/ОГЭ) или по словам пользователя (вступительные / другой). Без списка тем — не переходить к самооценке.

## Триггер
- После `exam-subject`
- До `exam-self-assess`

## Логика
1. Прочитать `users/<user_key>/profile.md` → `exam_type`, `exam_subject`, `exam_subject_variant`
2. Показать **список тем экзамена** (карта ниже) или попросить пользователя перечислить
3. Пользователь подтверждает список целиком, убирает лишнее или дописывает свои темы
4. **Записать в `users/<user_key>/profile.md` → `exam_topics: [тема1, тема2, ...]`**
5. Если пользователь дал свой список — `exam_topics_source: user|codifier|mixed`
6. Подтвердить и перейти к `exam-self-assess`

## Формат сообщения (общий)

> 📋 Вот **темы**, которые обычно входят в [exam_type] по [предмет]. Отметь, что актуально для тебя:
>
> [нумерованный список тем]
>
> Можешь:
> - написать **«все»** — если готовишься по полному кодификатору
> - перечислить **номера** тем, которые нужны (например: `1, 3, 5-7`)
> - **дописать** свои темы, если чего-то нет в списке
> - для своего экзамена — просто **перечислить темы** списком

### ЕГЭ — информатика (`ege` + `informatics`)

**Источник:** кодификатор ФИПИ 2025-2026 (обновлён 2026-06-19 на основе тестов сабагентов).

1. **Информация и её кодирование** — системы счисления, кодирование, объём информации, равномерное/неравномерное кодирование, **теория информации** (энтропия Шеннона)
2. **Логика и алгебра логики** — булевы функции, законы логики, предикаты
3. **Алгоритмы и программирование** — базовые конструкции, основы Python/C++, отладка
4. **Массивы, строки, файлы** — работа с коллекциями данных
5. **Рекурсия и сортировки** — рекурсивные алгоритмы, оценка сложности
6. **Графы и деревья** — обходы (DFS/BFS), кратчайшие пути (Дейкстра, Флойд), остовные деревья
7. **Динамическое программирование** — одномерное и двумерное ДП, оптимизация
8. **Базы данных и СУБД** — реляционная алгебра, SQL (SELECT/JOIN/GROUP BY), нормализация
9. **Компьютерные сети** — IP-адресация, маски подсети, протоколы (TCP/IP, HTTP/DNS), модель OSI
10. **Операционные системы** — процессы и потоки, файловые системы, управление памятью
11. **Моделирование и вычисления** — теория игр, клеточные автоматы, дискретная математика
12. **Задачи повышенной сложности (27)** — олимпиадные алгоритмы, NP-полные задачи, оптимизация

**Примечание для агента:** реальный кодификатор ФИПИ обновляется ежегодно. Перед онбордингом проверять актуальную версию на fipi.ru. **Если пользователь не назвал «все»** — попросить уточнить: «Полный кодификатор ФИПИ 12 блоков, или только те, что ты хочешь закрыть?» (важно, потому что задачи на теорию информации, БД, сети стабильно дают баллы, и пропуск = потеря).

### Вступительные / другой (`admission` | `other`)

> 📋 **Какие темы** будут на твоём экзамене? Перечисли списком — хоть коротко, хоть подробно.

## Поведение

| Ответ | Куда записать |
|---|---|
| `все` / `полный кодификатор` | весь список карты → `exam_topics` |
| Номера (`1, 3, 5`) | выбранные темы → `exam_topics` |
| Свой список тем | дословно в `exam_topics`, `exam_topics_source: user` |
| Карта + дополнения | объединить → `exam_topics_source: mixed` |
